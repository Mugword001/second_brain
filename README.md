# セカンドブレイン — Phase 1（撮る → 締切に）

プリントを撮って、締切を読み取り、確認して、カレンダーに入れる——アプリの主役の機能です。

## いちばん大事なこと：お金がかからない「お試しモード」付き

このアプリは、**AnthropicのAPIキーが無くても動きます**。

- キーが空のとき → **お試しモード**。見本の予定が表示され、「撮る→確認→カレンダー追加」の流れを全部試せます。お金は一切かかりません。
- キーを入れたとき → **本物モード**。撮った写真をClaudeが本当に読み取ります。

つまり、今日はキー無しで画面の流れを確認し、お母さん名義のキーが用意できたら入れるだけ、という順で進められます。

## 使い方（ローカル）

これまでと同じです。プロジェクトフォルダで：

```
npm install
npm run dev
```

http://localhost:3000 を開く → 「プリントを撮ってみる →」を押す → 写真を選ぶ → 読み取り結果を確認 → カレンダーに追加。

スマホで開くと、その場でカメラが起動して撮影できます（`capture` 対応）。

## 本物の読み取りを有効にするとき（お金がかかります）

⚠️ AnthropicのAPIは従量課金で、クレジットカード登録が必要です。契約・支払いは18歳以上が必須なので、**お母さん名義**でお願いします。

1. https://console.anthropic.com でアカウントを作り、支払い方法を登録。
2. APIキーを発行（`sk-ant-...` で始まる文字列）。
3. `.env.local` の `ANTHROPIC_API_KEY=` の右に貼り付けて保存。
4. `npm run dev` を再起動。

これで、お試しモードの黄色い付箋が消え、撮った写真が本当に読み取られるようになります。

料金の目安：プリント1枚の読み取りで数円以下。テストで数十枚試して数百円程度です。使った分だけなので、まずは少しずつ。

## カレンダー追加のしくみ

「カレンダーに追加する」を押すと、Googleカレンダーの予定作成画面が、タイトル・日付・メモが入った状態で開きます（企画書の「魔法のURL」方式）。あなたが内容を見て「保存」を押せば登録完了。アプリが勝手にカレンダーをいじることはないので安心です。

## まだ無いもの / これから

- 読み取った予定をSupabaseに保存して残す（今は登録するとカレンダー側に入るだけ）
- 撮った写真の保管
- ログイン（複数ユーザー対応）

これらはPhase 2以降で。まずは「撮る→締切に」の手応えを確かめてください。
